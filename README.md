🧘‍♀️ Yoga Posture Detection and Mood-Based Recommendation System

> Developed as part of an academic project under the guidance of **Prof. T. Senthil Kumar** (2024)

🌟 Project Overview

This project is a comprehensive mood-based Yoga Posture Detection and Recommendation System designed to promote personalized wellness. It includes a full-stack web application integrated with an NLP-powered chatbot, multimedia yoga resources, and a custom-built yoga pose dataset categorized by human moods.

🔧 Features

- 🧘‍♂️ Mood-Based Yoga Recommendations using a custom dataset of yoga poses mapped to emotional states
- 🤖 NLP Chatbot for guiding users through yoga practices and mood analysis
- 🎙️ Podcasts, 📹 Instructional Videos, and 📚 E-books for holistic learning
- 🧩 Full-stack web development with backend and frontend integration
- 🧠 Dataset development with mood classification and pose labeling
